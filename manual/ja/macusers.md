# 名前

macusers — AFP 経由で接続しているユーザをリストする

# 概要

**macusers**

`macusers [ -v | -version | --version | -h | -help | --help ]`

# 説明

**macusers**はAFP経由で接続しているユーザをリストする。

# オプション

**-v**, **-version**, **--version**

> バージョンを表示して終了する

**-h**, **-help**, **--help**

> ヘルプを表示して終了する
