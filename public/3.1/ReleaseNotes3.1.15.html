Netatalk 3.1.15

Netatalk 3.1.15 is available!

The Netatalk development team is proud to announce the latest release of the Netatalk 3.1 release series. Users are encouraged to update their servers to the 3.1 release series which is the stable and supported version for production systems.

What’s Changed

  • FIX: CVE-2022-43634 #186
  • FIX: CVE-2022-45188 #240
  • NEW: Support for macOS hosts, Intel and Apple silicon, #281
  • FIX: configure.ac: update deprecated autoconf syntax #331
  • UPD: configure.ac: Support linking with system shared libraries, #265 Introduces the –with-talloc option
  • FIX: macros: largefile-check macro for largefile (clang 16) #187
  • UPD: macros: Update pthread macro to the latest from gnu.org #273
  • FIX: initscripts: Modernize Systemd service file. #244
  • FIX: libatalk/conf: include sys/file.h for LOCK_EX #187
  • FIX: libatalk: Change log level for realpath() error, SF bug#666
  • FIX: libatalk: Change log level for real_name error, SF bug#596
  • FIX: libatalk: The my_bool type is deprecated as of MySQL 8.0.1, #129
  • UPD: libatalk: allow afpd to read read-protected afp.conf, SF bug#546
  • UPD: libatalk: Make the “valid users” option work in the Homes section, SF bug#449
  • UPD: libatalk: Check that FPDisconnectOldSession is successful, SF bug#634
  • UPD: libatalk: Bring iniparser library codebase in line with current version 4.1 #271 #282 #301
  • FIX: afpd: Provide MNTTYPE_NFS on OmniOS to make quota work, #117
  • FIX: afpd: Avoid triggering realpath() lookups with empty path, #277
  • FIX: spotlight: Spotlight searches can cause afpd to segfault, #56
  • UPD: spotlight: add support for tracker3, SF patch#147
  • FIX: macusers: Fix output for long usernames #219
  • FIX: macusers: account for usernames with non-word characters #219
  • FIX: macusers: Support NetBSD #219
  • FIX: Fix all function declarations without a prototype #272
  • FIX: Fix C99 compliance issues #294
  • FIX: Fix gcc10 compiler warnings #295
  • UPD: Remove acsiidocs sources and release notes script #227
  • FIX: manpages: afp.conf: Parameters are not quoted, SF bug#617
  • FIX: manpages: afp.conf: Document $u in home name, #123
  • FIX: manpages: afp.conf: Document the usage of guest user, #298
  • FIX: Document how the mysql cnid backend is configured, #69
  • FIX: Fix user-visible typos in log output and man pages. #218
  • FIX: Fix spelling, syntax, and dead URLs in html manual. #231
  • NEW: Create README.md #217
  • NEW: Set up GitHub workflow and static analysis with Sonarcloud #225
