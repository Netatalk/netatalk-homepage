Netatalk 2.2.9

Netatalk 2.2.9 is available!

The Netatalk development team is proud to announce the latest release of the Netatalk 2.2 release series. Users are encouraged to update their servers to the 2.2 release series which is the stable and supported version for production systems.

What’s Changed

  • [2.2] fix CVE-2022-45188 by @rdmark in https://github.com/Netatalk/netatalk/pull/254
  • [2.2] papd: Fix incorrect type in printer status check by @smagoun in https://github.com/Netatalk/netatalk/pull/320
  • [2.2] Improvements to the macusers script by @rdmark in https://github.com/Netatalk/netatalk/pull/263
  • [2.2] man pages: create an a2boot man page by @rdmark in https://github.com/Netatalk/netatalk/pull/235
  • [2.2] Improve systemd service dependencies, improving stability at boot on wifi only hosts by @rdmark in https://github.com/Netatalk/netatalk/pull/233
  • [2.2] Update manual to match current behavior and correct typos by @rdmark in https://github.com/Netatalk/netatalk/pull/230 https://github.com/Netatalk/netatalk/pull/234 https://github.com/Netatalk/netatalk/pull/257
  • [2.2] Remove release notes code since it’s no longer used by @rdmark in https://github.com/Netatalk/netatalk/pull/256
  • Create Github workflow that builds, tests, and runs static analysis by @rdmark in https://github.com/Netatalk/netatalk/pull/255 https://github.com/Netatalk/netatalk/pull/290 https://github.com/Netatalk/netatalk/pull/314
