Netatalk 2.2.8

Netatalk 2.2.8 is available!

The Netatalk development team is proud to announce latest release of the Netatalk 2.2 release series. Users are encouraged to update their servers to the 2.2 release series which is the stable and supported version for production systems.

What’s Changed

  • [2.2] uams: Remove OpenSSL 1.0 backwards compatibility header by @rdmark in https://github.com/Netatalk/netatalk/pull/183
  • [2.2] afpd: Unlink temp file after using it to check for EA support. by @rdmark in https://github.com/Netatalk/netatalk/pull/189
  • [2.2] afpd: Provide MNTTYPE_NFS on OmniOS by @rdmark in https://github.com/Netatalk/netatalk/pull/190
  • [2.2] afpd: reading from file may fail (backport from 3.x) by @rdmark in https://github.com/Netatalk/netatalk/pull/191
  • [2.2] asip-status.pl: Backport improvements to the asip-status.pl script by @rdmark in https://github.com/Netatalk/netatalk/pull/185
  • [2.2] afpd: Use daemon icon as the default for mounted volumes by @rdmark in https://github.com/Netatalk/netatalk/pull/184
  • [2.2] fix largefile-check macro for largefile with clang16 by @rdmark in https://github.com/Netatalk/netatalk/pull/192
  • [2.2] timelord: Fall back to timezone when tm_gmtoff is unavailable by @rdmark in https://github.com/Netatalk/netatalk/pull/195
  • [2.2] libatalk: Revert “Make the resource fork length equal to ad->ad_data” by @rdmark in https://github.com/Netatalk/netatalk/pull/199
  • [2.2] Disable quota by default; remove a few antiquated checks by @rdmark in https://github.com/Netatalk/netatalk/pull/200
  • [2.2] Add missing license based on BSD origin. by @rdmark in https://github.com/Netatalk/netatalk/pull/201
  • [2.2] Import Docbook documentation from the netatalk-doc repo by @rdmark in https://github.com/Netatalk/netatalk/pull/203
  • Fix a typo in debug message by @rdmark in https://github.com/Netatalk/netatalk/pull/208
  • [2.2] Fix spelling errors flagged by Debian’s lintian by @rdmark in https://github.com/Netatalk/netatalk/pull/209
  • [2.2] apple_dump: EA support in apple_dump script by @rdmark in https://github.com/Netatalk/netatalk/pull/204
  • [2.2] Update docbook scripts and readme by @rdmark in https://github.com/Netatalk/netatalk/pull/206
  • [2.2] Overhaul man page xml sources by @rdmark in https://github.com/Netatalk/netatalk/pull/210
  • [2.2] Overhaul html manual page sources by @rdmark in https://github.com/Netatalk/netatalk/pull/213
  • [2.2] afpd: Remove platform specific icons for unsupported OS’es. by @rdmark in https://github.com/Netatalk/netatalk/pull/222
  • [2.2] Enable DDP, timelord and a2boot by default by @rdmark in https://github.com/Netatalk/netatalk/pull/223
