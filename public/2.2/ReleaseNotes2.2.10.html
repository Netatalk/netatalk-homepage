Netatalk 2.2.10

Netatalk 2.2.10 is available!

The Netatalk development team is proud to announce the latest release of the Netatalk 2.2 release series. Users are encouraged to update their servers to the 2.2 release series which is the stable and supported version for production systems.

What’s Changed

  • [2.2] workflow: add libcups2-dev dependency by @rdmark in https://github.com/Netatalk/netatalk/pull/335
  • [2.2] Correct doc README by @rdmark in https://github.com/Netatalk/netatalk/pull/326
  • [2.2] Fix all warnings on running bootstrap by @dgsga in https://github.com/Netatalk/netatalk/pull/346
  • [2.2] bootstrap: Add checks for required packages by @rdmark in https://github.com/Netatalk/netatalk/pull/338
  • [2.2] Enable compilation on macOS hosts v2 by @dgsga in https://github.com/Netatalk/netatalk/pull/349
  • [2.2] macros: Use the AC_LANG_SOURCE macro, issue #347 by @rdmark in https://github.com/Netatalk/netatalk/pull/351
  • [2.2] Remove redundant documentation in conf files; improve man pages, #333 by @rdmark in https://github.com/Netatalk/netatalk/pull/334
  • [2.2] Use non-interactive PAM session when available by @rdmark in https://github.com/Netatalk/netatalk/pull/367
  • [2.2] papd: Future-proof CUPS API usage by @rdmark in https://github.com/Netatalk/netatalk/pull/373
  • [2.2] Update atalkd.service to be consistent with other init scripts. by @rdmark in https://github.com/Netatalk/netatalk/pull/372
  • [2.2] Adopt downstream Debian patches by @rdmark in https://github.com/Netatalk/netatalk/pull/370
  • [2.2] Restore tarball distribution of doc/ (GitHub#374) by @rdmark in https://github.com/Netatalk/netatalk/pull/375
  • [2.2] Regenerate man pages from xml sources by @rdmark in https://github.com/Netatalk/netatalk/pull/376

Full Changelog: https://github.com/Netatalk/netatalk/compare/netatalk-2-2-9…netatalk-2-2-10

Note: The Debian deb package distributed with this release is built for Debian 12 Bookworm.
