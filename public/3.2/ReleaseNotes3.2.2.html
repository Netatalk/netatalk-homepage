Netatalk 3.2.2

DOCKER COMPATIBILITY ALERT

If you previously used the Netatalk 3.1.18 Docker image for making Time Machine backups in macOS, you may run into errors after upgrading to the Netatalk 3.2.0 image or later. It is recommended to discard the backup sparsebundle and redo the backup. If you absolutely must access the older backup sparsebundle, downgrading to the 3.1.18 image should restore access.

Netatalk 3.2.2 is available!

The Netatalk team is proud to announce the latest version in the Netatalk 3.2 release series.

When using the Meson build system with the -Dwith-embedded-ssl flag, Netatalk now relies on a system OpenSSL or LibreSSL library as the provider of the CAST cryptography functions. At the same time, we remove source files that were reportedly incompatible with the redistribution terms of this package’s license. All users of the 3.2 release series are strongly recommended to upgrade their versions to 3.2.2.

What’s Changed

  • Create a security policy by @rdmark in https://github.com/Netatalk/netatalk/pull/1166
  • Remove obsolete Red Hat Upstart and SuSE SysV init scripts by @dgsga in https://github.com/Netatalk/netatalk/pull/1163
  • meson: Fix errors in PAM support macro by @dgsga in https://github.com/Netatalk/netatalk/pull/1178
  • meson: Fix perl shebang substitution in cnid2_create script by @dgsga in https://github.com/Netatalk/netatalk/pull/1183
  • meson: Add option to override system WolfSSL with embedded WolfSSL by @dgsga in https://github.com/Netatalk/netatalk/pull/1176
  • meson: Fix operation of D-Bus path macros by @dgsga in https://github.com/Netatalk/netatalk/pull/1182
  • Use external SSL dependency to provide cast header by @dgsga in https://github.com/Netatalk/netatalk/pull/1186
  • autotools: gcc 8.5 expects explicit library flags for libgcrypt by @rdmark in https://github.com/Netatalk/netatalk/pull/1188
  • meson: Fix errors in shadow password macro by @dgsga in https://github.com/Netatalk/netatalk/pull/1192

Full Changelog: https://github.com/Netatalk/netatalk/compare/netatalk-3-2-1…netatalk-3-2-2

Note: The Debian deb package distributed with this release is built for Debian 12 Bookworm.
