What is Netatalk?

Netatalk is a cross-platform open source implementation of AFP (Apple Filing Protocol), the protocol used for file sharing on Apple II, Classic Mac OS, and Mac OS X 10.0 ~ 10.8. With Mac OS X 10.9 onward, Apple made SMB the preferred file sharing protocol, but macOS Sonoma still ships with an AFP client.

Netatalk supports “AFP over TCP”, and if you use the legacy Netatalk v2 also “AFP over AppleTalk”. AppleTalk is required for file sharing with very old Macs as well as Apple IIe and IIGS computers, that don’t support AFP over TCP. Additionally, AppleTalk support brings the convenience of automatic service discovery, a printer server (papd) which can act as a two-way bridge for using modern printers on old Macs, a time server (timelord), and an Apple II netboot server (a2boot.)

You will see a mix of documentation for Netatalk v3 and v2 on this wiki.
